### 处理步骤

水样数据的处理步骤如下：  
  1. 全部以字符串的形式读取Analysis.txt。  
  2. 使用剔除的方式（缺省数据预处理方式）预处理数据，保存预处理后的数据集。  
  3. 进行数据摘要及数据可视化。  
  4. 允许用户选择其他预处理方式重新处理数据集，并重新进行可视化，对比新旧数据集。  

### 包含文件

- load_file.m 对原文进行读取
- main.m 主程序，运行获得各种数据摘要以及数据可视化、预处理结果
- output_file.m 输出文件
- preprocessing.m 数据预处理函数
- visualization.m 数据可视化函数
- Analytic_report.docx 分析报告
- Analytic_filled_by_XXX.txt 三个文件代表了除缺省的剔除方式外，所有的预处理后数据集
